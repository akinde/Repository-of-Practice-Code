<div class="markdown-content" id="problem-content">
<p>You are climbing a stair case and it takes <strong>A</strong> steps to reach to the top.</p>
<p>Each time you can either climb 1 or 2 steps. In how many distinct ways can you climb to the top?</p>
<p><br/></p>
<p><strong>Input Format:</strong></p>
<div class="highlighter-rouge"><pre class="highlight"><code>The first and the only argument contains an integer A, the number of steps.
</code></pre>
</div>
<p><strong>Output Format:</strong></p>
<div class="highlighter-rouge"><pre class="highlight"><code>Return an integer, representing the number of ways to reach the top.
</code></pre>
</div>
<p><strong>Constrains:</strong></p>
<div class="highlighter-rouge"><pre class="highlight"><code>1 &lt;= A &lt;= 36
</code></pre>
</div>
<p><strong>Example :</strong></p>
<p>Input 1:</p>
<div class="highlighter-rouge"><pre class="highlight"><code>A = 2 Output 1:

2 Explanation 1:

[1, 1], [2] Input 2:

A = 3 Output 2:

3 Explanation 2: 

[1 1 1], [1 2], [2 1]
</code></pre>
</div>

</div>
