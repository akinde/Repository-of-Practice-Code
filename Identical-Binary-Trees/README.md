<p>Two binary trees are considered equal if they are structurally identical and the nodes have the same value.</p>

<p><strong>Example :</strong></p>
<div class="highlighter-rouge"><pre class="highlight"><code>Input : 

   1       1
  / \     / \
 2   3   2   3

Output : 
  1 or True
</code></pre>
